## Params

<ParamField query="label" type="string" default="Text">
  Text for the text area label. The label appears in the panel in the Makeswift builder.
</ParamField>

<ParamField query="defaultValue" type="string">
  Defines the value shown in the text area in the Makeswift builder when the
  value isnâ€™t inputted.
</ParamField>

## Prop type

The TextArea control passes a `string` to your component. If you don't set a `defaultValue` and no value is set in the builder, your component receives `undefined`.

## Example

The following example adds a Text Area control to the `children` prop of a Paragraph component.

```tsx Paragraph.makeswift.ts
import { TextArea } from "@makeswift/runtime/controls"
import { runtime } from "@/lib/makeswift/runtime"

import { Paragraph } from "./Paragraph"

runtime.registerComponent(Paragraph, {
  type: "paragraph",
  label: "Paragraph",
  props: {
    children: TextArea({ label: 'Paragraph text', defaultValue: '' }),
  },
})
```
