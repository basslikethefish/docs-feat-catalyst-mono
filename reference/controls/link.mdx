## Params

<ParamField query="label" type="string" default="On click">
  Text for the panel label that appears in the Makeswift builder.
</ParamField>

## Prop type

The Link control passes an object of type `LinkValue` to your component, or `undefined` if you don't set a value.

```javascript
type LinkValue = {
  href: string
  target: '_blank' | '_self' | undefined
  onClick(event: MouseEvent<HTMLElement>): void
}
```

## Example

```jsx /components/Link/Link.makeswift.js
import { Link } from "@makeswift/runtime/controls"
import { runtime } from "@/lib/makeswift/runtime"

import { Img } from "./Img"

runtime.registerComponent(Img, {
  type: "link",
  label: "Link",
  props: {
    link: Link(),
  },
})
```
