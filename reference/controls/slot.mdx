## Params

The Slot control currently doesn't take any parameters.

<RequestExample>

```jsx /components/Box/Box.makeswift.js
import { Slot } from "@makeswift/runtime/controls"
import { runtime } from "@/lib/makeswift/runtime"

import { Box } from "./Box"

runtime.registerComponent(Box, {
  type: "box",
  label: "Box",
  props: {
    children: Slot(),
  },
})
```

</RequestExample>
