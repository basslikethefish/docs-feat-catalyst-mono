</ParamField>

## Types

### Image.Format

```javascript
type Image.Format = { 
  URL: string
  WithDimensions: ImageWithDimensions
}
``` 

| Property | Type | Description |
| --- | --- | --- |
| URL | string | Image URL. |
| WithDimensions | ImageWithDimensions | Includes image dimensions. | 

### ImageWithDimensions

```javascript
type ImageWithDimensions = {
  url: string
  dimensions: {
    width: number
    height: number
  }
}
```

| Property | Type | Description | 
| --- | --- | --- |
| url | string | Unique ID for the Image option. |
| dimensions.width | integer | Image width in pixels. | 
| dimensions.height | integer | Image height in pixels. | 



<RequestExample>

```jsx /components/Img/Img.makeswift.js
import { Image } from "@makeswift/runtime/controls"
import { runtime } from "@/lib/makeswift/runtime"

import { Img } from "./Img"

runtime.registerComponent(Img, {
  type: "image",
  label: "Upload an Image",
  props: {
    src: Image(),
    image: Image({ format: Image.Format.WithDimensions }),
  },
})
```

</RequestExample>
