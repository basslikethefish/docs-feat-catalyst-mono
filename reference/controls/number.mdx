## Params

<ParamField query="label" type="string" default="Number">
  Text for the number label. The label appears in the panel in the Makeswift
  builder.
</ParamField>

<ParamField query="labelOrientation" type="LabelOrientation" default="horizontal">
  Orientation of the number label within the panel.
</ParamField>

<ParamField query="defaultValue" type="number" default="undefined">
  Defines the value shown for the number in the Makeswift builder when the
  value isn’t set. If you don’t provide a default value, the Checkbox control
  passes an `undefined` value to your component.
</ParamField>

<ParamField query="min" type="number" default="0">
  The smallest number that can be set in the panel input.
</ParamField>

<ParamField query="min" type="number" default="0">
  The largest number that can be set in the panel input.
</ParamField>

<ParamField query="step" type="number" default="1">
  Amount by which to increment the number value when using the ↑ ↓ arrows to drag the panel input.
</ParamField>

<ParamField query="suffix" type="string" default="">
  Decorative text appended to the end of the panel input.
</ParamField>


## Types

```javascript LabelOrientation
type LabelOrientation = 'vertical' | 'horizontal'
```

<RequestExample>

```jsx /components/NumberComponent/NumberComponent.makeswift.js
import { Number } from "@makeswift/runtime/controls"
import { runtime } from "@/lib/makeswift/runtime"

import { NumberComponent } from "./NumberComponent"

runtime.registerComponent(NumberComponent, {
  type: "number-component",
  label: "NumberComponent",
  props: {
    defaultValue: Number({ label: 'Default value', defaultValue: 3 }),
    min: Number({ label: 'Min', defaultValue: 0 }),
    max: Number({ label: 'Max', defaultValue: 10 }),
  },
})
```
